# Project Overview

**Goal:** Build a shared grocery list with real-time updates, simple UI, and secure authentication.

**Core Features**
- Account creation & login
- Shared household lists (add/edit/delete)
- Real-time sync via Supabase Realtime
- Responsive Flutter front-end

**Non-Goals (Phase 1)**
- Payments
- Predictive recommendations
